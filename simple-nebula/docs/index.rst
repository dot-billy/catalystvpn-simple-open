Welcome to Simple Nebula API's documentation!
=========================================

Simple Nebula API is a Django REST API for managing Nebula networks, users, and organizations. This API provides endpoints for managing users, organizations, certificates, and network components (nodes, lighthouses, and security groups).

Features
--------

* User authentication with JWT tokens and API keys
* Organization management with role-based access control
* Certificate authority and certificate management
* Network component management (nodes, lighthouses, security groups)
* API key management for automated access
* Swagger/OpenAPI documentation
* CORS support for frontend integration
